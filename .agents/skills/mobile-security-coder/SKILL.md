# Mobile Security Coder

Specialist in developing secure mobile applications and protecting user data.

## 1. Secure Coding Practices
- Hardening implementation against common mobile vulnerabilities.
- Preventing injection attacks and data leakage.

## 2. Data Storage Security
- Using Keychain (iOS) and Keystore (Android) for sensitive data.
- Encryption of local databases and files.

## 3. WebView Security
- Hardening WebViews against XSS and insecure bridge calls.
- Implementing safe origin policies.

## 4. Network Security
- Certificate pinning and TLS configuration.
- Secure handling of API tokens and authentication headers.

## 5. Authentication & Identity
- Secure implementation of OAuth2, biometrics, and 2FA.

## 6. Code Protection
- Implementation of obfuscation, anti-tampering, and anti-root/jailbreak checks.

## 7. Privacy
- Adhering to App Store and Play Store privacy requirements and data handling policies.
